# `figures/`

Generated by `notebooks/thp_implementation.ipynb`; regenerated on each full run.

- **`01_causal_masking.png`** -- attention weights with and without causal masking, side by side.
- **`02_temporal_encodings.png`** -- sinusoidal, Time2Vec (linear and periodic components shown separately), and log-gap encodings over the same irregular timestamp sequence.
- **`03_true_hawkes_intensity.png`** -- the ground-truth intensity curve for the simulated Hawkes process, showing the jump-then-decay pattern after each exciting event.
- **`04_training_curve.png`** -- train vs validation NLL over training, with the true data-generating process's own NLL plotted as a reference line.
- **`05_true_vs_predicted_intensity.png`** -- true vs model-predicted intensity on a held-out test sequence.

These are committed so the README and notebook render correctly on GitHub without anyone needing to re-run the notebook first.
